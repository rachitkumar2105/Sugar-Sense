About Our AI Model

State-of-the-art deep learning architecture for diabetes prediction

Neural Network Architecture

  • Input Layer: 8 clinical features
  • Hidden Layer 1: 128 neurons + BatchNorm + ReLU
  • Hidden Layer 2: 256 neurons + Dropout (0.3)
  • Hidden Layer 3: 128 neurons + BatchNorm
  • Hidden Layer 4: 64 neurons + Dropout (0.2)
  • Output Layer: 1 neuron + Sigmoid activation

Training Details

  • Dataset: 100,000 patient records
  • Framework: PyTorch 2.9.1
  • Optimizer: Adam with learning rate optimization
  • Loss Function: Binary Cross-Entropy
  • Regularization: Dropout + BatchNormalization
  • Validation: Cross-validated on test set

Clinical Features

  • Demographics: Age, Gender
  • Medical History: Hypertension, Heart Disease
  • Lifestyle: Smoking History
  • Biometrics: BMI (Body Mass Index)
  • Lab Results: HbA1c Level, Blood Glucose
  • Preprocessing: Label Encoding + Standard Scaling
